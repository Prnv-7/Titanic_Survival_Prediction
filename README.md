# Titanic_Survival_Prediction
Data Dictionary:

* survived - (0 - No, 1 - Yes)
* pclass - Ticket class
* Sex - Sex
* Age - Age in Years
* Sibsp - No: of siblings/spouses aboard the Titanic
* Parch - No: of parents/children aboard the Titanic
* Ticket - Ticket Number
* Fare - Passenger Fare
* Cabin - Cabin Number
* Embarked - Port of embarkation
